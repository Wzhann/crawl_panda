# Breathe He — ESP32-S3 舵机动作控制器

基于 ESP32-S3 的 Wi-Fi 遥控舵机控制器，内嵌 Web 控制面板，用于控制玩偶的**呼吸点头**和**爬行**动作。

## 项目概览

| 项目 | 说明 |
|------|------|
| **芯片** | ESP32-S3 |
| **舵机** | 3 路 180° 舵机 (PWM 50Hz) |
| **Wi-Fi** | AP 模式，SSID: `paxing_hx`（无密码） |
| **IP** | `192.168.4.1` |
| **框架** | ESP-IDF v5.5 (FreeRTOS) |
| **语言** | C++ (C++17)，单文件 `main.cc` |
| **编译产物** | `action_cat.bin` |

## 硬件引脚

### 电源管理

| 引脚 | 功能 | 说明 |
|------|------|------|
| **IO7** | `POWER_CTRL` | 电源锁存 — HIGH 保持供电，LOW 断电 |
| **IO6** | `POWER_OUT` (ADC1_CH5) | 电源按钮 — ADC 值 >1241 为松开，≤1241 为按下 |
| **IO3** | 电池电压 (ADC1_CH2) | 分压比 `Rup=2k / Rlow=4.7k`，分压系数 ×2 |

### 舵机 (3 × 180°)

| 引脚 | 名称 | 中立角 | 范围 | 说明 |
|------|------|--------|------|------|
| **IO4** | `SERVO_POWER` | — | — | 舵机电源使能 |
| **IO15** | Head 头部 | 90° | 0°~180° | 0°=左, 90°=中, 180°=右 |
| **IO16** | LeftHand 左手 | 90° | 0°~180° | 0°=上, 90°=休息, 180°=爬下 |
| **IO17** | RightHand 右手 | 90° | 0°~180° | 0°=爬下, 90°=休息, 180°=上 (左手镜像) |

## 启动流程

```
app_main()
  ├─ nvs_flash_init()          NVS 初始化
  ├─ InitPower()               锁存 IO7 HIGH，启动电源监控任务
  ├─ InitServos()              3 路舵机归中 (90°)，使能 IO4 供电
  ├─ InitAutoRun()  ← 宏控制   启动自运行动画任务
  ├─ InitWiFi()                启动 AP: paxing_hx
  └─ StartHttpServer()         启动 HTTP 服务 (192.168.4.1)
```

## 编译宏

| 宏 | 默认值 | 作用 |
|----|--------|------|
| `ENABLE_AUTO_RUN` | `1` | 编译自运行动画功能（含 `/api/autoplay` 端点） |
| `AUTO_RUN_DEFAULT_ON` | `1` | 上电后动画默认开启 (1) 或暂停 (0) |

## FreeRTOS 任务

### `power_mon` — 电源监控

- 每 **20ms** 轮询 IO6 ADC，对称迟滞消抖（50ms）
- **长按 1.5 秒** → IO7 拉 LOW，系统断电
- 每 **5 秒** 采样电池电压 32 次过采样 + 一阶低通滤波
- 电量百分比: `(Vbat - 3.2V) / (4.2V - 3.2V) × 100%`

### `auto_run` — 自运行动画 (ENABLE_AUTO_RUN=1 时编译)

- 每 **20ms** 一帧
- **Head**：正弦点头，中心 90° ± 20°，周期 1.5s
- **双手爬行**：sin² 缓动扫掠，LH 90°↔30°，RH 90°↔180°
- 每 **8s** 切换交替/同时爬行模式，切换时有 1s smoothstep 淡入淡出
- 暂停时只 sleep，不碰舵机

## PWM 舵机控制

- **频率**：50Hz（周期 20ms）
- **脉冲范围**：500μs（0°）~ 2500μs（180°）
- **分辨率**：13-bit
- **映射公式**：`pulse_us = 500 + angle × 2000 / 180`

## HTTP API

| 方法 | 路径 | 说明 |
|------|------|------|
| `GET` | `/` | Web 控制面板 |
| `POST` | `/api/servo` | 设置舵机 `{"angles":[head, lh, rh]}` |
| `GET` | `/api/battery` | 电池状态 `{"voltage_mv":xxx,"level":xx}` |
| `GET/POST` | `/api/autoplay` | 查询/切换自运行动画 (需 `ENABLE_AUTO_RUN=1`) |

## Web 控制面板

内嵌固件中的单页 Web 应用 (`192.168.4.1`)：

| 模块 | 功能 |
|------|------|
| **头部控制** | 滑块 0°~180°，快捷按钮（左/中/右），点头动画 |
| **左手/右手** | 独立滑块 + 快捷按钮 |
| **双手摇杆** | Canvas 触摸摇杆，Y=爬行深度，X=转弯侧重，松手回弹 |
| **爬行控制** | 交替/同时/单臂 4 种模式，起止角度可调，频率 0.05~3.0 Hz |
| **快捷预设** | 全部休息、爬行姿势、前进/左右转一键执行 |
| **自运行动画** | 🤖 按钮开关 ESP32 内置动画 (需编译宏开启) |
| **动作录制** | 100ms 间隔录制 3 通道，生成 C++ 代码片段，支持回放 |
| **电池显示** | 电量百分比 + 电压，5s 刷新 |

## 开发

idf.py -p /dev/ttyUSB0 flash

## 设计要点

1. **长按关机**：硬件锁存电路，短按开机、长按 1.5s IO7 拉 LOW 断电
2. **镜面对称**：左右手 90° 均为休息位，运动方向相反（LH 角度↑=爬，RH 角度↓=爬）
3. **PWM 精度**：13-bit @ 50Hz，角度精度约 0.22°
4. **电池监测**：ADC 曲线拟合校准 + 32×过采样 + 指数移动平均
5. **单文件架构**：所有逻辑 + HTML/JS 集中在 `main.cc`，便于维护
6. **手动优先**：网页手动控制舵机时自动暂停自运行动画，避免控制冲突
